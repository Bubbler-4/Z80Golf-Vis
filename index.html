Planned Features

  1. Support basic one-byte instructions.
  2. Show debug information: registers, memory and disassembly.
  3. Support prefixed instructions (IX/IY-related, bit-related, extra-op).
  4. Support undocumented instructions based on this article.
  5. Support accurate flags and the R register (for random source).
  6. Someday move to GitHub Pages and add support for permalink.
